== Description ==

Rouh is a sophisticated WordPress theme designed for modern blogs and news websites. It features a clean, minimalist design with excellent typography and a focus on readability. The theme includes a unique compact slider for featured posts, multiple layout options, and comprehensive customization features.

Key Features:
* Responsive design that works on all devices
* Accessibility-ready with proper keyboard navigation and screen reader support
* Dark mode toggle with smooth transitions
* Compact slider for featured posts
* Custom color schemes with CSS custom properties
* Right sidebar layout
* Custom logo support
* Post formats support
* Threaded comments
* Custom menu support
* Translation ready
* SEO optimized
* Fast loading with optimized CSS and JavaScript
* Social media integration
* Instagram feed integration
* Related posts functionality
* Author cards
* Custom widgets
* Multiple navigation options
* Skip-to-content functionality for better accessibility

The theme uses modern web technologies including CSS Grid, Flexbox, and CSS custom properties for maximum flexibility and performance.

== Installation ==

1. Upload the theme files to the `/wp-content/themes/rouh` directory, or install the theme through the WordPress themes screen directly.
2. Activate the theme through the 'Themes' screen in WordPress.
3. Use the Customizer to configure the theme settings.
4. Create your menus in Appearance > Menus.
5. Add widgets to the sidebar in Appearance > Widgets.

== Frequently Asked Questions ==

= Does this theme support Gutenberg? =

Yes, Rouh includes comprehensive Gutenberg block styles and is fully compatible with the WordPress block editor.

= Can I use this theme for a business website? =

While Rouh is primarily designed for blogs and news websites, its clean design makes it suitable for various types of content-focused websites.

= Does the theme support custom headers? =

Yes, you can set a custom header image through the WordPress Customizer.

= Is the theme translation ready? =

Yes, Rouh is fully translation ready and includes a .pot file for translators.

= Does the theme support RTL languages? =

Yes, Rouh includes RTL support for right-to-left languages.

= How do I enable dark mode? =

The theme includes an automatic dark mode toggle button that appears in the header. Users can switch between light and dark modes.

= Can I customize the colors? =

Yes, the theme uses CSS custom properties that can be customized through the WordPress Customizer or by child theme modifications.

== Privacy Policy ==

Rouh does not collect any user data by default. However, the theme may use the following third-party services:

* Google Fonts: The theme may load fonts from Google Fonts. Google's privacy policy applies: https://policies.google.com/privacy
* Gravatar: For displaying user avatars in comments. Automattic's privacy policy applies: https://automattic.com/privacy/

If you use any additional plugins or services with this theme, please refer to their respective privacy policies.
